# HTTP `Hypertext Transfer Protocol`

HTTP operates on a client-server model
and is the primary method by which web browsers and servers communicate
to share information over the internet.

While its main purpose is to transfer webpages and provide other resources during web browsing,
it is also able to transfer data,
facilitating file sharing.

When a user enters a website domain and tries to access it,
HTTP establishes a connection to the server hosting the domain and provides access to the website.
For example, when a user types a domain name, such as `google.com`,
into their browser, HTTP connects to the web server hosting that domain.
The web server then responds by sending the HTML content
or the code that defines the structure and content of the webpage.

Another form of HTTP is `HTTP Secure`.
HTTPS can encrypt a user's HTTP requests and webpages,
providing greater network security and preventing common cybersecurity threats,
such as `man-in-the-middle` attacks.

HTTPS is more widely used than HTTP
because of its improved security features,
and most major browsers now only support HTTPS.
