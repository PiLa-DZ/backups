# Why use GEMINI-CLI

- Auto `fix typo`
- Write bast `git commit message`
- Finding `BUGs` is better than you.

## Examples

```bash
# FIX: db.js
git commit -m "fix(db): handle empty file crash and add json formatting"

# WARN: Fixing the Typos in Git History

# FIX: task-class.js refactor:
git commit -m "refactor(task): use static factory method and optimize updates"

# FIX: Inside update(id, description)

# FIX: app.js --> Normalize the input.
# FIX: app.js --> Handling the Async Nature
git commit -m "feat(cli): implement command routing and argument parsing"

# FIX: help.js help menu:
git commit -m "feat(cli): add formatted help menu and usage guide"

# TEST: You have to re-write another test from scratch
```
