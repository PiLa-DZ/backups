# 🎓 The Final Roadmap Review

Look at how beautifully everything maps out in your mind now:

1. **REST API:** The architectural guidelines (Nouns for paths, HTTP Verbs for actions, statelessness).
2. **JSON APIs:** The strict ruleset on how to design keys inside those JSON payloads.
3. **SOAP:** The legacy, armored XML protocol for bulletproof corporate financial compliance.
4. **gRPC:** The lightning-fast internal server-to-server microservices protocol using binary Protobuf.
5. **GraphQL:** The single-gateway graph language that lets the client choose its own data shape.
6. **OpenAPI Spec:** The ultimate machine-readable blueprint language used to fully automate the documentation and testing of your REST APIs.
